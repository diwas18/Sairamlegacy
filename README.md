# Sairam Chasma Pasal

A full-featured e-commerce platform developed to transform a traditional eyewear shop into a modern online business.

## Features

* Product browsing with detailed descriptions and images
* Product rating and review system
* Virtual try-on feature for better user experience
* Online payment and Cash on Delivery options
* Order tracking system
* Email notifications for offers and updates
* Eye checkup booking system
* Admin dashboard for managing products, users, and orders (CRUD operations)

## Technologies Used

* Laravel (PHP Framework)
* MySQL Database
* JavaScript
* HTML/CSS

## Purpose

This project was developed to digitize a traditional eyewear shop, providing customers with a convenient and interactive online shopping experience while improving business operations.
